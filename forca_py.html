Projeto Python: Jogo da Forca

Vamos criar um clássico: o Jogo da Forca! O computador escolhe uma palavra ao acaso e tu tens de adivinhar as letras antes que as tuas tentativas acabem.

Bloco 1: Configuração e Escolha da Palavra

Primeiro, importamos a biblioteca random para o computador conseguir escolher palavras diferentes em cada jogo.

import random

# Lista de palavras para o jogo
palavras = ["computador", "programacao", "python", "escola", "matematica", "teclado", "janela"]

# Escolher uma palavra ao acaso
palavra_secreta = random.choice(palavras)
letras_descobertas = []
tentativas = 6

# Criar a lista visual da palavra (ex: _ _ _ _)
for letra in palavra_secreta:
    letras_descobertas.append("_")

print()
print("--- JOGO DA FORCA ---")

Bloco 2: O Ciclo Principal e Tentativas

Aqui usamos um ciclo while. O jogo continua enquanto o jogador tiver tentativas e ainda existirem letras escondidas ("_").

while tentativas > 0 and "_" in letras_descobertas:
    print("\n" + " ".join(letras_descobertas))
    print(f"Tentativas restantes: {tentativas}")
    
    palpite = input("Digita uma letra: ").lower()

    if palpite in palavra_secreta:
        print(f"Boa! A letra '{palpite}' existe na palavra.")
        # Atualizar as letras descobertas
        for i in range(len(palavra_secreta)):
            if palavra_secreta[i] == palpite:
                letras_descobertas[i] = palpite
    else:
        tentativas = tentativas - 1
        print(f"Erraste! A letra '{palpite}' não existe.")

Bloco 3: Resultado Final

Finalmente, verificamos se o jogador conseguiu descobrir todas as letras ou se as tentativas esgotaram.

if "_" not in letras_descobertas:
    print("\nPARABÉNS! Ganhaste o jogo!")
    print(f"A palavra era: {palavra_secreta}")
else:
    print("\nFicaste sem tentativas!")
    print(f"A palavra secreta era: {palavra_secreta}")

print("Até à próxima!")
Notas para a Aula:
  • Variáveis: Usamos tentativas para controlar o estado do jogo.
  • Lógica: O comando "".join() ajuda a mostrar a lista de letras de forma bonita.
  • Desafio: Tenta adicionar uma verificação para não perderes tentativas se repetires uma letra que já tinhas dito!
