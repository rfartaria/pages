Projeto Python: Jogo da Forca Progressivo

Nesta versão, a forca não aparece toda de uma vez. Ela é construída linha a linha à medida que erras!

Bloco 1: A Lista de Linhas

Cada item na lista HANGMANPICS representa agora uma linha horizontal do desenho.
import random HANGMANPICS = [' +---+', ' +---+', ' | |', ' O |', ' |', ' |', ' |', '========='] palavras = ["python", "escola", "janela", "teclado"] secreta = random.choice(palavras) descobertas = ["_"] * len(secreta) erros = 0 while erros < len(HANGMANPICS) and "_" in descobertas: print() # Imprime as linhas da forca de acordo com os erros for i in range(erros + 1): print(HANGMANPICS[i]) print("\nPalavra:", " ".join(descobertas)) palpite = input("Letra: ").lower() if palpite in secreta: for i in range(len(secreta)): if secreta[i] == palpite: descobertas[i] = palpite else: erros = erros + 1 if "_" not in descobertas: print("\nGanhaste! A palavra era:", secreta) else: print("\nPERDESTE!") for linha in HANGMANPICS: print(linha)
