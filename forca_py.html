Projeto Python: Jogo da Forca

Cria um jogo onde tens de adivinhar a palavra secreta letra a letra!

Bloco 1: Preparar a Palavra

import random palavras = ["python", "escola", "teclado", "matematica"] secreta = random.choice(palavras) descobertas = [] tentativas = 6 for letra in secreta: descobertas.append("_")

Bloco 2: O Ciclo de Jogo

O jogo corre enquanto tiveres tentativas e ainda houver "_" por preencher.
while tentativas > 0 and "_" in descobertas: print(" ".join(descobertas)) palpite = input("Letra: ").lower() if palpite in secreta: for i in range(len(secreta)): if secreta[i] == palpite: descobertas[i] = palpite else: tentativas -= 1 print("Erraste! Restam:", tentativas)
