Projeto Python: O Cercado Perfeito

Nesta atividade, vamos ajudar um jardineiro a descobrir todas as formas possíveis de construir um canteiro retangular usando exatamente 24 metros de rede, para encontrar a que tem a maior área.

[Image of a rectangular garden plot with dimensions width and length]

Bloco 1: Configuração e Variáveis Iniciais

Primeiro, definimos o nosso perímetro e calculamos a "soma mágica" (metade do perímetro), que é o que a largura e o comprimento devem somar.

# 1. Definir o perímetro total da rede
perimetro_total = 24
metade_perimetro = perimetro_total / 2

# Começamos com a largura mínima (1 metro)
largura = 1

print()
print("--- EXPLORAÇÃO: O JARDINEIRO ---")
print(f"Rede disponível: {perimetro_total} metros")

Bloco 2: O Ciclo de Cálculo (while)

Aqui usamos um ciclo while para testar as várias larguras possíveis. O ciclo continua enquanto a largura for menor ou igual ao comprimento para não repetirmos retângulos.

print()
print("================================")
print("Listagem de dimensões possíveis:")

# Calculamos o comprimento inicial
comprimento = metade_perimetro - largura

while largura <= comprimento:
    # Calcular a área do retângulo atual
    area = largura * comprimento
    
    print(f"Largura: {largura}m | Comprimento: {comprimento}m | Área: {area}m2")
    
    # Passo importante: passamos para a próxima largura
    largura = largura + 1
    
    # Atualizamos o comprimento para a nova largura
    comprimento = metade_perimetro - largura
