Projeto Python: O Jardineiro (Perímetro 60m)

Nesta atividade, vamos usar um perímetro de 60 metros e uma variável de controlo para decidir quando parar de calcular.

Bloco 1: Variáveis e Estado Inicial

Definimos a variável continuar como True para o ciclo começar a funcionar.
perimetro = 60 largura = 0 continuar = True

Bloco 2: O Ciclo de Exploração

O símbolo // garante que o resultado da divisão seja um número inteiro.
while continuar: largura = largura + 1 comprimento = perimetro // 2 - largura area = largura * comprimento print("Largura:", largura, "| Comprimento:", comprimento, "| Área:", area) # Se a largura chegar ao valor do comprimento, paramos if largura >= comprimento: continuar = False
Dica de Matemática: Repara que com 60m de rede, a maior área possível é 225m² (quando o canteiro é um quadrado de 15x15).
