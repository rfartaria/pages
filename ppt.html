Projeto Python: Jogo com Pontuação

Nesta versão, vamos adicionar um Placar e um comando para Sair do jogo.

Bloco 1: Configuração e Variáveis do Placar

import random # 1. Definir as jogadas jogadas = ["pedra", "papel", "tesoura"] # Variáveis para guardar os pontos vitorias_utilizador = 0 vitorias_computador = 0 empates = 0 print() print("--- JOGO: PEDRA, PAPEL OU TESOURA ---")

Bloco 2: O Ciclo Principal e Opção Sair

Aqui usamos um while True dentro de outro para garantir que o jogo continue até pedires para sair.
while True: # Loop para jogar várias vezes while True: # Loop para validação da jogada print() print("================================") print("Podes jogar: pedra, papel ou tesoura.") jogada_utilizador = input("Escreve a tua escolha (ou 'sair'): ") jogada_utilizador = jogada_utilizador.lower() # Verificar se o utilizador quer terminar o programa if jogada_utilizador in ["sair", "exit", "q", "x"]: print("\nObrigado por jogar! Até à próxima.\n") exit() if jogada_utilizador in jogadas: break else: print("Escolha inválida. Tenta novamente.")

Bloco 3: A Escolha do Computador

jogada_computador = random.choice(jogadas) print() print("Tu escolheste:", jogada_utilizador) print("O computador escolheu:", jogada_computador)

Bloco 4: Lógica de Vitória e Atualização do Placar

if jogada_utilizador == jogada_computador: print("Resultado: Empataram!") empates += 1 elif jogada_utilizador == "pedra" and jogada_computador == "tesoura": print("Resultado: Tu ganhaste!") vitorias_utilizador += 1 elif jogada_utilizador == "papel" and jogada_computador == "pedra": print("Resultado: Tu ganhaste!") vitorias_utilizador += 1 elif jogada_utilizador == "tesoura" and jogada_computador == "papel": print("Resultado: Tu ganhaste!") vitorias_utilizador += 1 else: print("Resultado: O computador ganhou!") vitorias_computador += 1

Bloco 5: Mostrar o Placar Final

print() print("--- PLACAR ATUAL ---") print("Vitórias Utilizador:", vitorias_utilizador) print("Vitórias Computador:", vitorias_computador) print("Empates:", empates)
